# Zenzic VS Code Extension

This is the official VS Code extension for the Zenzic Language Server. It acts as a lightweight client connecting Visual Studio Code to the `zenzic lsp` backend.

## Requirements

You must have Zenzic installed. You can install it via:

```bash
uv tool install zenzic
```

## Configuration

- `zenzic.executablePath`: Path to the Zenzic executable or virtual environment binary. Default is `"zenzic"`.
